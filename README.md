# Atividade TDD Java

Este projeto é uma atividade da matéria **Técnicas de Programação II**, do curso de **Desenvolvimento de Software Multiplataforma (DSM)** da **Fatec Votorantim**.

## Tecnologias

- Java 11
- Maven
- Spring Boot 2.7.18

## Como rodar o projeto

### Pré-requisitos

- Java 11 instalado
- Maven instalado

### Passos

1. Clone o repositório e acesse a pasta do projeto.
2. No diretório raiz, execute:

```bash
mvn spring-boot:run
```

A aplicação será iniciada localmente.

## Como rodar os testes

Para executar os testes automatizados:

```bash
mvn test
```
